# UI Home Style Previews Design

## Goal

Add full homepage mockups to the existing `/ui-design` preview page so the playful K1-K6 visual directions can be compared at a glance.

## Scope

- Use the existing `playfulStyles` data in `prototype/src/pages/UIDesignPreviewPage.vue`.
- Add one homepage preview per K1-K6 playful style.
- Give each playful style its own homepage structure, not a shared template with different colors.
- Do not change the real `/` homepage route.
- Do not change the shared app top navigation.

## Design

The new section lives in `/ui-design` after the compact horizontal previews. It expands the playful style board into six homepage sketches:

- K1: bookmark garden with flower-bed site cards.
- K2: hand-drawn sticky-note workspace.
- K3: pixel handheld game dashboard.
- K4: magazine cover and column layout.
- K5: resource forest with growth paths.
- K6: URL galaxy with orbital site markers.

## Confirmation

The user confirmed K6 "URL galaxy / space exploration" as the final design style and color direction on 2026-04-27.
